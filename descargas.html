En este apartado te encontraras con los comandos para descargar y/o actualizar las diferentes aplicasiones que utilizaremos para poder correr nuetra aplicasion desde Ubuntu



Actualizar paquetes


1. La siguiente instruccion ( sudo update ) sirve para identificar lo que ya estan instalados y lo que se debe actualizar en el ubuntu.

2. La siguiente instruccion ( sudo upgrate ) como lo dice el codigo sirve para actualizar cualquier paquete. Te pedira oprima la tecla ( s ) para confirmar la actualizaci√≥n.


MYSQL


1. Obligatoriamente tiene que estar conectado el dispositivo a internet. ( apt search mysql ) aparecera un listado de todas las apliacaciones que tiene instaladas.

2. Segundo paso (sudo apt install mysql-server ) y oprime la tecla enter para que comieze la descarga.

3. De la instruccion ( s ) para afirmar la descarga.


Apache


1. Tecleara el siguiente codigo ( apt search apache ) aparecera un listado de todas las apliacaciones que tiene instaladas.

2. Ya terminado el primer paso dara la instruccion (sudo apt install apache2) para que comieze la descarga.

3. Dara la instruccion ( s ) para afirmar la descarga.

4. En este paso aprendera a identificar si esta funcionando apache ( service --status-all | grep apache2 ) si aparece un [+]es que esta corriendo apache y si aparece [-]es que no esta corriendo.


PHP


1. Repetira el mismo procedimiento ( apt search php ) aparecera un listado de todas las apliacaciones que tiene instaladas.

2. Ya terminado el primer paso dara la instruccion (sudo apt install php) para que comieze la descarga.

3. Dara la instruccion ( s ) para afirmar la peticion y comieze la descarga.
