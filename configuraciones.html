Configuracion del VirtualHosts


Lo primero que tenemos que hacer es entrar a la carpeta (/etc/apache2/sites-available) ,una vez en esa carpeta escribiremos (sudo nano library.com.conf) para proceder a modificarlo.
Dentro del archivo, escribe el codigo que se encuentra a continuacion:

VirtualHost*:80>
Server Name library.com
Server Alias www.library.com
Server Admin cristianramirez01@gmail.com
DocumentRoot /var/log/library.com-error.log
CustomLog /var/log/library.com-acces.log. common
ServerName
/VirtualHost>
Revisa que tus carpetas estén en el siguiente orden (/etc/apache2/sites-available/)escribiendo el comando (ls). En la carpeta (sites-available) escribe: (sudo a2ensite library.com )para habilitar el dominio. Reinicia el servicio de apache con el comando (sudo service apache2 restart). Entra a la carpeta (/etc/hosts/) y agrega lo siguiente:

127.0.0.1 library.com
127.0.0.1 www.library.com

Guarda los cambios mediante Ctrl+O
Una vez fuera de la maquina virtual en consola entra a la siguiente carpeta: (/etc/hosts) y agrega el ip de la red anfitrion + nombre del dominio.
Por ejemplo:

192.168.56.101 library.com
192.168.56.101 www.library.com
